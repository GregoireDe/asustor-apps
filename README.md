# Asustor Apps


Apps for Asustor OS (ADM 5.X)

- FlareSolverr
- Prowlarr
- Extended Settings

Building: `python3 apkg-tools_py3.py create <pkg_directory> [<destination_directory>]`

The folder contains APK apps if you want to install manually.
