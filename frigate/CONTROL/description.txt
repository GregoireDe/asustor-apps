Frigate is an open source NVR built around real-time AI object detection.
All processing is performed locally on your own hardware, and your camera feeds never leave your home.

Note:
1. The docker version is maintained by the developer.
2. Default port: 18970 (HTTP unauthenticated) / 18971 (HTTPS authenticated). Be carefull if you expose on Internet
3. Recommendation: Use only unauthenticated access to configure users after installation
